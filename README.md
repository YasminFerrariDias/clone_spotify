# Clone Spotify

Protótipo da página inicial do Spotify, desenvolvido durante a Imersão Front-End da Alura.

## Tecnologias

- [HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML) — estruturação da página
- [CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS) — estilização e layout
- [JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) — interatividade

## Sobre

Este projeto reproduz visualmente a página inicial do Spotify. Os botões ainda não possuem funcionalidade — o foco foi a construção da interface. Certificado da imersão disponível no [LinkedIn](https://www.linkedin.com/in/seu-perfil).
